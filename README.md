# Usage

## Follow steps below

1. Clone git project

   > git init

   > git clone https://github.com/gadaDev/blog-personal

2. Install project

   > npm install

3. Script

   | Script        |             Are              |
   | ------------- | :--------------------------: |
   | npm run dev   | Start app in dev environment |
   | npm run start |   Build and start project    |
   | npm run build |        Build project         |
